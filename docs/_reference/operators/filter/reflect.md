# reflect

Category: filter



Reflects space across a plane.

Can optionally expose which side of the plane a point is on as an iteration value for upstream ops.

## Parameters

* `Enable` *Enable*
* `Direction` *Direction*
  * `custom` *Custom*
  * `xpos` *X+*
  * `xneg` *X-*
  * `ypos` *Y+*
  * `yneg` *Y-*
  * `zpos` *Z+*
  * `zneg` *Z-*
* `Planenormal` *Plane Normal*
* `Offset` *Offset*
* `Shift` *Shift*
* `Exposeiteration` *Expose Iteration*
* `Inspect` *Inspect*
* `Help` *Help*
