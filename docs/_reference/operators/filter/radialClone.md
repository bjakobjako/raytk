# radialClone

Category: filter
OP Type: raytk.operators.filter.radialClone



Repeats an SDF radially around an axis, combining the resulting shapes.

Note that this runs its input multiple times, which can lead to performance issues.

## Parameters

* `Enable`
* `Axis`
* `Count`
* `Anglerange`
* `Angleoffset`
* `Radiusoffset`
* `Mergetype`
* `Mergeradius`
* `Iterationtype`
* `Inspect`
