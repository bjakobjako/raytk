# boxFrameSdf

Category: sdf



SDF for the squared frame of the edges of a box.

## Parameters

* `Translate` *Translate*: Move the center of the shape.
* `Scale` *Scale*: The size of the box.
* `Thickness` *Thickness*: The thickness of the bars of the box.
* `Inspect` *Inspect*
