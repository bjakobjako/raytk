# planeSdf

Category: sdf



An infinite plane on the x, y, or z axis.

## Parameters

* `Axis` *Axis*: Which axis the plane faces.
  * `x` *X*
  * `y` *Y*
  * `z` *Z*
* `Offset` *Offset*: Shifts the plane forwards or backwards along the axis that it faces.
* `Inspect` *Inspect*
* `Help` *Help*
