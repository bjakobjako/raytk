# pyramidSdf

Category: sdf



A pyramid with four sides.

## Parameters

* `Translate` *Translate*: Moves the center of the base of the pyramid.
* `Height` *Height*: The height of the pyramid.
* `Width` *Width*: The width of the base of the pyramid.
* `Inspect` *Inspect*
* `Help` *Help*
