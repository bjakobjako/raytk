# axisDistanceField

Category: field
OP Type: raytk.operators.field.axisDistanceField



A float field that provides the distance from a specific point along a single axis.

## Parameters

* `Axis`
* `Center`
* `Coordtype`
* `Contexttype`
* `Inspect`
