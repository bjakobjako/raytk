# multiPointDistanceField

Category: field
OP Type: raytk.operators.field.multiPointDistanceField



A vector field that provides the distance from 4 specific points in space (one for each part of the vector).

## Parameters

* `Point1`
* `Point2`
* `Point3`
* `Point4`
* `Coordtype`
* `Contexttype`
* `Inspect`
