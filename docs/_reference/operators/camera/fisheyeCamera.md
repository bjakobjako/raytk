# fisheyeCamera

Category: camera
OP Type: raytk.operators.camera.fisheyeCamera



A 360 fisheye camera, that shows all directions from a specific point in space.

## Parameters

* `Fisheyemode`
* `Aperture`
* `Campos`
* `Camrot`
* `Lookatpos`
* `Camup`
