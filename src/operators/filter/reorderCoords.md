Swaps axes for the input.

## Parameters

* `Enable`
* `Axisx`: Which axis should be used as the new X axis.
  * `x`
  * `y`
  * `z`
* `Axisy`: Which axis should be used as the new Y axis.
  * `x`
  * `y`
  * `z`
* `Axisz`: Which axis should be used as the new Z axis.
  * `x`
  * `y`
  * `z`
